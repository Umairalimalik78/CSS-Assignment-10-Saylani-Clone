## ✨ Features

- 🏠 Upwork-style Home / Landing page
- 🎨 Pixel-perfect Upwork UI clone
- 📱 Responsive and clean layout
- 🧭 Navigation bar with Upwork branding
- 🔍 Hero section with search functionality UI
- 💼 Freelancer/Category showcase section
- 🌐 Built with pure HTML & CSS — no frameworks

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**

```bash
git clone https://github.com/Umairalimalik78/Upwork-Clone.git
```

2. **Navigate to the project folder**

```bash
cd Upwork-Clone
```

3. **Open `index.html` in your browser**

Simply double-click the `index.html` file or open it with **Live Server** in VS Code.
